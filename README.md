
cloud-neard
============

cloud-neard - node.js version - demonstrates the Neard NFC APIs.


Install
-------

    npm install cloudeebus


Requirements
------------

  * [neard](http://git.kernel.org?p=network/nfc/neard.git)
  * [Cloudeebus](https://github.com/01org/cloudeebus)
